Programmin Language Processor/PL/0' virtual machine assembly language programming

Excercise 1

Execute the following program add.s(2) by pl0das.pl, and understand the execution process.

start:
	lit,2     ; push 2.
	lit,3     ; push 3.
	opr,add   ; pop two constants 2 and 3, and push the value of '2+3'.
	opr,wrt   ; pop a value and output the value.
	opr,wrl   ; output a new line code.
	jmp,start ; jump to the address 0 to terminate this program.

Exercise 2

Modify 'add.s' so that it can output 1 if the value of "2+3" is even or 0 if if the value is odd.

Hints: Check the following instructions in the page A PL/0' virtual machine.

  • opr,odd
  • jpc

Answer: even_odd.s(3)
