Programming Language Processor/Installation of a PL/0' compiler

How to install a PL/0' compiler

Download pl0d.tgz(27) into your directory and take the following steps in a terminal program on Linux.

% tar zxvf pl0d.tgz
% cd pl0d
% make

You can download the original compiler from http://www.k.hosei.ac.jp/~nakata/oCompiler/PL0compiler/prog_proc1.html , but we use a modified version of the compiler.

How to use pl0d

A command 'pl0d' can not only compile and execute a PL/0' program, but also output a compiled assembly program. In both cases, the command outputs errors in the form of HTML.

Compile and execute a PL/0' program

If a PL/0' program is given without any option, pl0d compiles and executes the given program.

In the following example, pl0d compiles and executes fact.pl0. It also outputs fact.pl0.html which shows how pl0d recognized fact.pl0.

% ./pl0d fact.pl0
; start compilation
; 1 errors
; start execution
1 1 
2 2 
3 6 
4 24 
5 120 
6 720 
7 5040 
8 40320 
9 362880

Output a compiled assembly language

pl0d outputs a compiled assembly language program if a PL/0' program is given with an option '-l' (the lower case of L).

% ./pl0d -l fact.pl0
; start compilation
; 1 errors

; code
L000: jmp,L016
L001: jmp,L002
